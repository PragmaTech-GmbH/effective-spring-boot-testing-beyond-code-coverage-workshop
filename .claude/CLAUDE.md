# Instructions for Slides

Whenever your work within `slides/` and make changes to a `.md` file, consider the following:

- Each time you create a diagram with excalidraw, save it as a `.excalidraw` file in the `assets/` folder. Export it as well as `.png` in the `assets/` folder. For the filename, add the prefix of the lab the diagram refers to, e.g. `lab-1-context-caching.png` and `lab-1-context-caching.excalidraw`.
